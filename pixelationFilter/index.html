Colors are expressed as 3-bit sequences, one bit per color channel.

You can type anything you like in the "Input" panel and it will immediately be displayed in the image above.

When you click "Filter" your input bits will be run through the three lines of code you write in the "newRed", "newGreen" and "newBlue" boxes. You can type anything you like. The variables "red", "green", and "blue" will automatically be set to either 0 or 1 for you on each pixel.

You cannot edit the Output bits directly but you can still clean them up or view them in their raw form.

Happy filtering!
